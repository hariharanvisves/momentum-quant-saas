
# Momentum Quant SaaS

## Features
- Live NIFTY 500 Scanner
- Weekly Automated Rebalancer
- Zerodha Kite Auto Execution (Toggle)
- Multi-factor Optimizer
- Portfolio Advisory SaaS Ready Structure

## Run

npm install
npm start

Open:
http://localhost:3000

## Enable Kite
Set environment variables in .env

KITE_API_KEY=your_key
KITE_ACCESS_TOKEN=your_token

Toggle execution in UI.
